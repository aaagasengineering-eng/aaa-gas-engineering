## HOW TO ADD YOUR IMAGES

1. Copy your images into the correct folder above
2. Name them EXACTLY as shown (lowercase, no spaces)
3. Open index.html in VS Code
4. Find the comment that says "REPLACE" — uncomment the img tag
5. Save and refresh browser

### Image recommended sizes:
- Logo:         200×200px, .png (transparent background)
- Hero bg:      1920×1080px, .jpg
- Products:     800×600px, .jpg
- Factory:      1200×800px, .jpg
- Certifications: 200×100px, .png

### Compress images first (free):
→ https://squoosh.app

---

## HOW TO EDIT CONTENT

- **Company info, text**: Edit `index.html` directly in VS Code
- **Colors / fonts**: Edit `css/style.css` — change CSS variables at the top
- **Add new section**: Add HTML in `index.html` + CSS in `style.css`
- **Form behavior**: Edit `js/form.js`

## HOW TO HOST ON VERCEL (free)

1. Create account at https://github.com
2. Create new repository, upload this folder

4. Select your repo — click Deploy
5. Website is live! Connect your domain from Namecheap
